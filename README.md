# Emotion Based Recommendation System

This project detects facial emotion and recommends meals, movies, and music based on the detected emotion.

The system uses computer vision and APIs to provide personalized recommendations.

## Features

• Emotion detection using facial image  
• Movie recommendation using OMDB API  
• Meal recommendation using Spoonacular API  
• Music recommendation using Spotify API  
• Streamlit web application interface  
• Webcam image capture option  

## Technologies Used

Python  
OpenCV  
DeepFace  
Streamlit  
Spotify API  
OMDB API  
Spoonacular API  

## Files

main.py → Console version using webcam  
app.py → Streamlit web application  

## Run Streamlit App

streamlit run app.py
